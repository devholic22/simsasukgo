# simsasukgo

Google Maps-based travel planning service that visualizes bookmarked locations and real-time business hours.

## Project Description

When traveling abroad, tracking multiple visited places or bookmarked locations can be challenging. **simsasukgo** solves this by:

- Displaying all bookmarked locations on a map
- Showing real-time business operating status
- Providing an intuitive visualization of your travel itinerary
